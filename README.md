# Personal Portfolio Website

This is a simple personal portfolio website created using HTML and CSS.  
The project consists of two pages: a Home page and a Contact page.

## Features

### Home Page
- Name
- Role / Profession
- Short Description
- Interests Section
- Projects Section

### Contact Page
- Name Input Field
- Email Input Field
- Message Textarea
- Submit Button

### Additional Features
- Instagram Link
- LinkedIn Link
- Navigation Between Pages

## Technologies Used
- HTML5
- CSS3

## Purpose of the Project

This project was created to practice basic front-end web development concepts such as:
- HTML Structure
- Forms
- Page Navigation
- CSS Styling
- Portfolio Layout Design

## Pages Included
1. Home Page
2. Contact Page
